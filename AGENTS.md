# AGENTS

## Baseline
- Unity version baseline: Unity 6.3 LTS (use 6000.3 documentation).
- Input: Use the Input System package APIs only. Do not use `UnityEngine.Input` or other legacy input APIs.
- Packages: Keep package versions pinned; do not change `Packages/manifest.json` or package lock files unless a ticket explicitly requires it.
- Safety: All hacking content must remain fictional/simulated; avoid real-world network or exploit behavior.
